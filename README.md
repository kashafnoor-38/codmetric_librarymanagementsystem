# 📚 Library Management System

A Java Swing-based Library Management System developed as **Task 3** for the **Codmetric Java Programming Internship**.

## 📖 Description

This application allows users to manage books in a library through a simple and user-friendly graphical interface. It is built using **Java Swing** for the GUI and **ArrayList** for storing book records.

## ✨ Features

- ➕ Add a new book
- 📕 Borrow a book
- 📗 Return a book
- 📋 Display all books
- ❌ Error handling using dialog boxes
- 🖥️ Interactive Java Swing GUI

## 🛠️ Technologies Used

- Java
- Java Swing
- ArrayList
- Eclipse IDE

## 📂 Project Structure

```
Codmetric_Task3

```

## ▶️ How to Run

1. Open the project in Eclipse IDE.
2. Run the `Main.java` file.
3. Use the graphical interface to:
   - Add books
   - Borrow books
   - Return books
   - View the library catalog
